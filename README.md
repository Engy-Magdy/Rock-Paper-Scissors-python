# Rock Paper Scissors Game 👊🏻📄✂️

A simple command-line Rock Paper Scissors game built with Python.

## Features
- The player chooses rock, paper, or scissors.
- The computer makes a random choice.
- The game determines the winner according to the rules.
- Includes ASCII art for a better user experience.

## Technologies Used
- Python
- Random module

## How to Run
1. Download the project files.
2. Open the project folder.
3. Run the following command:

```bash
python main.py
```
